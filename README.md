# ASEAN-7 Gold × CPU Analysis

This repository provides replication code and data for the study on ASEAN-7 stock markets.

## Data
- cpu.xlsx

## Code
- CPU_ASEAN.R

## Methods
- OLS (Newey–West HAC)
- GARCH-X (AR(1)-GARCH(1,1))
- Subsample analysis (Full, Post-2005, Post-2015)
